# Linear Regression Visualization

This Python script performs linear regression on user-input data and visualizes the fitting line using the matplotlib library.

## Usage

1. Run the script.
2. Input the independent variables (X), dependent variables (y), and the range for the plot.
3. Provide labels for the X and Y axes.

## Dependencies

- Python 3.x
- NumPy
- Matplotlib

## Installation

1. Install Python: [Python Official Website](https://www.python.org/downloads/)
2. Install NumPy: `pip install numpy`
3. Install Matplotlib: `pip install matplotlib`

## How to Run

```bash
python linear_regression_visualization.py
```

## Input Instructions

- When prompted, enter the number of elements for independent and dependent variables.
- Input each element one by one.

## Output

The script will display a scatter plot of the input data along with the fitting line generated by linear regression.

## File Structure

- `linear_regression_visualization.py`: Main script file.
- `readme.txt`: This file providing information about the script.
